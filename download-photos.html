Download Site Images

Click the button. It will fetch all 39 photos and package them into one ZIP file — then you upload that ZIP's contents to GitHub. Takes about 1–2 minutes.

Starting…

After the ZIP downloads:

  1. Find site-images.zip in your Downloads folder and extract / unzip it — you'll get a folder called images with 39 JPG files.
  2. Go to github.com/taimlee/Doorsandmoldings → site → images
  3. Click Add file → Upload files
  4. Open the extracted images folder and select all 39 JPGs (Ctrl+A), then drag them into GitHub.
  5. Scroll down and click Commit changes.
  6. Wait 1–2 minutes for Netlify to deploy — photos will appear on the site.
